Recetas Inclusivas API

API REST con Express, MongoDB, bcrypt y JSON Web Token.

Información de la API

Esta API permite registrar usuarios, iniciar sesión, proteger rutas con JWT y gestionar recetas pensadas para personas con restricciones alimentarias.

Endpoints principales

  • POST /api/users/register
  • POST /api/users/login
  • GET /api/users
  • GET /api/recipes
  • GET /api/recipes/:id
  • POST /api/recipes
  • GET /api/reviews
  • POST /api/reviews
